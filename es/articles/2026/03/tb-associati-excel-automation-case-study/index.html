De 50 días a 1,5: cómo T&B Associati automatizó la migración de Excel al software de gestión con IA

Caso de estudio: T&B Associati redujo la migración de datos de Excel al software de gestión de proyectos de 50 jornadas-hombre a 1,5 usando la IA privada de HT-X. 52.000 horas introducidas, 180% de ROI.

7 March 2026 · 4 min read
caso de estudioautomatizacióngestión de proyectosExcelpymes

El cliente: T&B Associati

T&B Associati es una firma de consultoría especializada en gestión de proyectos complejos. Con un equipo de profesionales que coordinan recursos en múltiples encargos simultáneamente, la empresa depende de software de gestión de proyectos para planificar actividades, asignar recursos y hacer seguimiento de las horas trabajadas.

El problema: Excel como cuello de botella

Como en muchas pymes europeas, el flujo de trabajo operativo de T&B Associati tenía un punto crítico: los datos llegaban en formato Excel.

Clientes, proveedores y colaboradores externos enviaban hojas de cálculo con horas trabajadas, recursos desplegados y actividades realizadas. Estos datos debían transferirse manualmente al sistema de gestión de proyectos, un proceso que requería:

  • Abrir y analizar decenas de archivos Excel con formatos diferentes
  • Interpretar las entradas y mapearlas a los campos del software de gestión
  • Introducir manualmente miles de filas de datos
  • Verificar para prevenir errores
  • Rellenar campos derivados y calculados

Para un solo proyecto de gran escala, esta operación requería 50 jornadas-hombre — casi dos meses y medio de una persona trabajando a tiempo completo. Un coste enorme en términos de tiempo, recursos y riesgo de error humano.

La solución: automatización inteligente con PRISMA

HT-X diseñó una solución a medida basada en PRISMA (Private Intelligence Stack for Modular AI), la infraestructura de IA privada que ejecuta modelos de inteligencia artificial on-premise, sin que los datos de la empresa salgan del perímetro de la organización.

Cómo funciona

El sistema desarrollado para T&B Associati opera en tres fases:

1. Análisis del software de gestión

HT-X estudió el software de gestión de proyectos en uso, mapeando la estructura de datos, campos obligatorios, relaciones entre entidades (proyectos, recursos, actividades, horas) y reglas de validación.

2. Análisis y normalización de datos Excel

Las hojas Excel entrantes tenían formatos variables: columnas diferentes, nomenclatura inconsistente, datos faltantes o parciales. El sistema de IA analiza cada archivo, reconoce la estructura, identifica las columnas relevantes y normaliza los datos en un formato uniforme.

3. Población inteligente del sistema de gestión

Una vez normalizados, los datos se introducen automáticamente en el software de gestión. El sistema no solo copia valores: rellena inteligentemente campos derivados, resuelve ambigüedades en nombres de recursos, asocia códigos de proyecto y calcula totales. Cada entrada se verifica contra las reglas de validación del sistema.

Por qué PRISMA

Elegir una solución basada en PRISMA garantiza que:

  • Los datos se quedan en la empresa: ningún archivo Excel ni dato de proyecto se envía a servidores externos
  • Cumplimiento del RGPD: el procesamiento ocurre completamente dentro de la infraestructura del cliente
  • Personalización total: el sistema fue construido según los requisitos específicos de T&B Associati
  • Independencia: sin dependencia de proveedores, sin costes por token

Resultados

La primera migración del proyecto piloto produjo números que hablan por sí solos:

La cifra más significativa: 1,5 días frente a 50 jornadas-hombre. Una operación que paralizaba a un equipo durante semanas ahora se completa en un día y medio, con una calidad de datos superior a la entrada manual.

Qué significa esto para las pymes europeas

El caso de T&B Associati ilustra un patrón común en las empresas europeas: procesos críticos que siguen dependiendo de Excel como formato de intercambio de datos. Cada vez que un operador abre una hoja de cálculo y copia datos a otro sistema, hay una oportunidad de automatización.

Las soluciones de IA on-premise como las basadas en PRISMA permiten:

  • Automatizar flujos de datos repetitivos sin exponer información sensible
  • Reducir drásticamente los tiempos operativos
  • Eliminar errores de transcripción
  • Liberar recursos para actividades de mayor valor

No necesitas reemplazar tu software de gestión. No necesitas cambiar cómo los clientes envían datos. Necesitas un puente inteligente que conecte ambos mundos — y eso es exactamente lo que la IA privada hace mejor.
