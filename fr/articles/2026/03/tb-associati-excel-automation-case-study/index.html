De 50 jours à 1,5 : comment T&B Associati a automatisé la migration Excel vers le logiciel de gestion avec l'IA

Étude de cas : T&B Associati a réduit la migration des données d'Excel vers un logiciel de gestion de projets de 50 jours-homme à 1,5 grâce à l'IA privée de HT-X. 52 000 heures saisies, 180 % de ROI.

7 March 2026 · 4 min read
étude de casautomatisationgestion de projetExcelPME

Le client : T&B Associati

T&B Associati est un cabinet de conseil spécialisé dans la gestion de projets complexes. Avec une équipe de professionnels coordonnant des ressources sur plusieurs missions simultanément, l’entreprise s’appuie sur un logiciel de gestion de projets pour planifier les activités, allouer les ressources et suivre les heures travaillées.

Le problème : Excel comme goulot d’étranglement

Comme dans de nombreuses PME européennes, le flux de travail opérationnel de T&B Associati avait un point critique : les données arrivaient au format Excel.

Clients, fournisseurs et collaborateurs externes envoyaient des feuilles de calcul avec les heures travaillées, les ressources déployées et les activités réalisées. Ces données devaient être transférées manuellement dans le système de gestion de projets, un processus nécessitant :

  • L’ouverture et l’analyse de dizaines de fichiers Excel aux formats différents
  • L’interprétation des entrées et leur correspondance avec les champs du logiciel de gestion
  • La saisie manuelle de milliers de lignes de données
  • Des vérifications croisées pour prévenir les erreurs
  • Le remplissage des champs dérivés et calculés

Pour un seul projet de grande envergure, cette opération nécessitait 50 jours-homme — près de deux mois et demi de travail à temps plein d’une personne. Un coût énorme en temps, en ressources et en risque d’erreur humaine.

La solution : automatisation intelligente avec PRISMA

HT-X a conçu une solution sur mesure basée sur PRISMA (Private Intelligence Stack for Modular AI), l’infrastructure d’IA privée qui fait fonctionner des modèles d’intelligence artificielle on-premise, sans que les données de l’entreprise ne quittent le périmètre de l’organisation.

Comment ça fonctionne

Le système développé pour T&B Associati opère en trois phases :

1. Analyse du logiciel de gestion

HT-X a étudié le logiciel de gestion de projets utilisé, cartographiant la structure des données, les champs obligatoires, les relations entre entités (projets, ressources, activités, heures) et les règles de validation.

2. Analyse et normalisation des données Excel

Les feuilles Excel entrantes avaient des formats variables : colonnes différentes, nommage incohérent, données manquantes ou partielles. Le système IA analyse chaque fichier, reconnaît la structure, identifie les colonnes pertinentes et normalise les données dans un format uniforme.

3. Remplissage intelligent du système de gestion

Une fois normalisées, les données sont automatiquement saisies dans le logiciel de gestion. Le système ne se contente pas de copier des valeurs : il remplit intelligemment les champs dérivés, résout les ambiguïtés dans les noms de ressources, associe les codes projet et calcule les totaux. Chaque entrée est vérifiée par rapport aux règles de validation du système.

Pourquoi PRISMA

Le choix d’une solution basée sur PRISMA garantit que :

  • Les données restent en interne : aucun fichier Excel ni donnée de projet n’est envoyé à des serveurs externes
  • Conformité RGPD : le traitement s’effectue entièrement au sein de l’infrastructure du client
  • Personnalisation totale : le système a été construit selon les exigences spécifiques de T&B Associati
  • Indépendance : pas de dépendance fournisseur, pas de coûts par token

Résultats

La migration du premier projet pilote a produit des chiffres qui parlent d’eux-mêmes :

Métrique Avant Après Amélioration
Temps de migration 50 jours-homme 1,5 jour -97 %
Heures saisies manuelles 52 000 automatiques
Cellules remplies manuelles 23 000 automatiques
Erreurs de saisie fréquentes quasi nulles réduction drastique
ROI — 180 % sur le premier projet

Le chiffre le plus significatif : 1,5 jour contre 50 jours-homme. Une opération qui paralysait une équipe pendant des semaines se termine désormais en un jour et demi, avec une qualité de données supérieure à la saisie manuelle.

Ce que cela signifie pour les PME européennes

Le cas de T&B Associati illustre un schéma commun à travers les entreprises européennes : des processus critiques qui dépendent encore d’Excel comme format d’échange de données. Chaque fois qu’un opérateur ouvre une feuille de calcul et copie des données dans un autre système, il y a une opportunité d’automatisation.

Les solutions IA on-premise comme celles basées sur PRISMA permettent de :

  • Automatiser les flux de données répétitifs sans exposer d’informations sensibles
  • Réduire drastiquement les temps opérationnels
  • Éliminer les erreurs de transcription
  • Libérer des ressources pour des activités à plus forte valeur ajoutée
