Von 50 Tagen auf 1,5: wie T&B Associati die Migration von Excel zur Managementsoftware mit KI automatisiert hat

Fallstudie: T&B Associati hat die Datenmigration von Excel zur Projektmanagement-Software von 50 Personentagen auf 1,5 reduziert — mit der privaten KI von HT-X. 52.000 Stunden erfasst, 180% ROI.

7 March 2026 · 3 min read
FallstudieAutomatisierungProjektmanagementExcelKMU

Der Kunde: T&B Associati

T&B Associati ist eine Beratungsgesellschaft, die auf komplexes Projektmanagement spezialisiert ist. Mit einem Team von Fachleuten, das Ressourcen ueber mehrere Auftraege gleichzeitig koordiniert, setzt das Unternehmen auf Projektmanagement-Software zur Planung, Ressourcenzuweisung und Zeiterfassung.

Das Problem: Excel als Engpass

Wie in vielen europaeischen KMU hatte der operative Workflow von T&B Associati einen kritischen Schwachpunkt: Daten kamen im Excel-Format an.

Kunden, Lieferanten und externe Mitarbeiter schickten Tabellen mit geleisteten Stunden, eingesetzten Ressourcen und durchgefuehrten Aktivitaeten. Diese Daten mussten manuell in das Projektmanagementsystem uebertragen werden:

  • Oeffnen und Analysieren Dutzender Excel-Dateien mit verschiedenen Formaten
  • Interpretation der Eintraege und Zuordnung zu den Feldern der Managementsoftware
  • Manuelles Eintragen Tausender Datenzeilen
  • Gegenpruefen zur Fehlervermeidung
  • Ausfuellen abgeleiteter und berechneter Felder

Fuer ein einziges Grossprojekt erforderte dieser Vorgang 50 Personentage. Ein enormer Aufwand an Zeit, Ressourcen und mit hohem Risiko menschlicher Fehler.

Die Loesung: intelligente Automatisierung mit PRISMA

HT-X hat eine massgeschneiderte Loesung auf Basis von PRISMA (Private Intelligence Stack for Modular AI) entwickelt — der privaten KI-Infrastruktur, die KI-Modelle On-Premise ausfuehrt, ohne dass Unternehmensdaten den Organisationsbereich verlassen.

So funktioniert es

Das fuer T&B Associati entwickelte System arbeitet in drei Phasen:

1. Analyse der Managementsoftware

HT-X hat die eingesetzte Projektmanagement-Software untersucht: Datenstruktur, Pflichtfelder, Beziehungen zwischen Entitaeten (Projekte, Ressourcen, Aktivitaeten, Stunden) und Validierungsregeln.

2. Analyse und Normalisierung der Excel-Daten

Die eingehenden Excel-Tabellen hatten variable Formate: unterschiedliche Spalten, inkonsistente Bezeichnungen, fehlende oder unvollstaendige Daten. Das KI-System analysiert jede Datei, erkennt die Struktur, identifiziert relevante Spalten und normalisiert die Daten in ein einheitliches Format.

3. Intelligente Befuellung des Managementsystems

Nach der Normalisierung werden die Daten automatisch in die Managementsoftware eingetragen. Das System kopiert nicht nur Werte: Es fuellt intelligent abgeleitete Felder aus, loest Mehrdeutigkeiten bei Ressourcennamen auf, ordnet Projektcodes zu und berechnet Summen.

Warum PRISMA

Die Wahl einer PRISMA-basierten Loesung stellt sicher, dass:

  • Daten im Unternehmen bleiben: Keine Excel-Dateien oder Projektdaten werden an externe Server gesendet
  • DSGVO-Konformitaet: Die Verarbeitung erfolgt vollstaendig in der Infrastruktur des Kunden
  • Vollstaendige Anpassung: Das System wurde auf die spezifischen Anforderungen von T&B Associati zugeschnitten
  • Unabhaengigkeit: Kein Vendor Lock-in, keine Kosten pro Token
